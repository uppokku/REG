[데이터 기반 Trade Off 분석] 저는 특정 공정이 시스템 전체에 미치는 영향을 이해할 수 있습니다.
프로세스 전반을 바라보는 시각으로, 설계와 생산 사이에서 활약하는 공정 엔지니어가 되겠습니다.
전국 반도체 회로 설계 경진대회에서 아날로그 LIF뉴런 네트워크를 설계해 2위를 수상하였습니다.
Virtuoso를 이용해 센싱, 서밍, 모션 3가지 뉴런 층을 설계하였습니다. 아날로그 LIF회로는 노이즈에 취약한 특성이 있습니다.
특히 앞 단인 센싱과 서밍에 inhabition을 적용하면 미약한 신호가 노이즈로 보이는 위험이 있습니다.
이에 신호가 충분히 확보되는 모션에만 반대 방향 신호가 서로를 억제하는 승자독식 방식을 적용 하였습니다.
또한 뉴런의 출력단에 있는 슈미트 트리거 구조 3가지를 비교분석 하였습니다. 370.2E-15W로 가장 낮은 전력을 가지는 구조는 네트워크 구성 시 신호가 소멸되는 문제가 있었습니다.
전력과 신뢰성 간의 트레이드 오프를 고려해, 787.7e-15W구조를 선택하였습니다. [파이프라인 최적화] 파이프라인의 어느 단계에 어떤 유닛을 배치하느냐가 성능을 결정하듯, Metal공정에서도 배선 구조가 Critical Path를 결정합니다.
시스템을 설계하는 시각으로 Critical Path를 개선하는 엔지니어가 되겠습니다. RISC-V기반 파이프라인 CPU 최적화 프로젝트를 진행 하였습니다.
목표는 FPGA보드 위에서 작동하는 Quick Sort의 실행 속도를 개선하는 것이 었습니다. Control Hazard 감지 유닛의 위치를 MEM단계에서 ID단계로 변경하였습니다.
해당 유닛이 MEM단계에 있을 경우, 분기를 실행할 때 이미 파이프라인에 들어온 3개의 명령어를 버려야 했습니다.
대신에 ID단계에서 일찍이 분기 명령어를 감지하고 잘못된 명령어를 빠르게 삭제하도록 구조를 변경했습니다.
그 결과, Control Hazard로 인한 성능 손실을 기존 3사이클에서 1사이클로 줄여 알고리즘을 크게 개선할 수 있었습니다.
