[Trade Off 분석을 통한 최적화]
저는 특정 설정이 시스템 전체에 미치는 영향을 이해할 수 있습니다. 프로세스 전반을 바라보는 시각으로, 회로와 시스템 사이에서 활약하는 항공 엔지니어가 되겠습니다.
전국 반도체 회로 설계 경진대회에서 아날로그 LIF뉴런 네트워크를 설계해 2위를 수상하였습니다. Virtuoso를 이용해 센싱, 서밍, 모션 3가지 뉴런 층을 설계하였습니다. 아날로그 LIF회로는 노이즈에 취약한 특성이 있습니다. 특히 앞 단인 센싱과 서밍에 inhabition을 적용하면 미약한 신호가 노이즈로 보이는 위험이 있습니다. 이에 신호가 충분히 확보되는 모션에만 반대 방향 신호가 서로를 억제하는 승자독식 방식을 적용 하였습니다.
또한 뉴런의 출력단에 있는 슈미트 트리거 구조 3가지를 비교분석 하였습니다. 370.2E-15W로 가장 낮은 전력을 가지는 구조는 네트워크 구성 시 출력 신호가 소멸되는 문제가 있었습니다. 전력과 신뢰성 간의 트레이드 오프를 고려해, 787.7e-15W구조를 선택하였습니다.
회로 조건 변화에 따른 신호 거동을 정량적으로 분석하고 최적화한 강점을, 항공 시스템 회로의 신호 무결성 검증에 활용하겠습니다.
